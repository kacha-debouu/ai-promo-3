# ai-promo-3 · BioPath flow + gallery

Auto-looping promos for product flows, recreated from Figma.

- **`/`** — Gallery: tiles linking to each animation
- **`/biopath/`** — BioPathAI · RNA Sequence Review (13-state agent workflow loop)

Sister project: [ai-promo-2](https://github.com/kacha-debouu/ai-promo-2) — boards.so iPhone promo (Vite + React + framer-motion).

## Run locally

It's static HTML — open `index.html` in a browser, or:

```bash
python3 -m http.server 5181
# then http://localhost:5181/
```

## Deploy

This is a plain static site — drop it on any host. Live preview: see Cloudflare Pages deploy.
